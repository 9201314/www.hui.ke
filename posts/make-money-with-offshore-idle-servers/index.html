    利用境外闲置服务器赚钱

    2022-11-19
    2023-09-29
    1 min read
    IT
    Linux
    Hits

      最近甲骨文发公告称将会开始关停闲置的永久免费实例，刚好又无意间发现了两个（一个已经无了，仅剩一个）国外的网赚平台，可以通过 Docker 运行，然后利用服务器闲置的资源换取一点点微薄的收入（真的很微薄，实测一天都赚不到一美分😂）。但是总归聊胜于无……既能提高甲骨文服务器的资源占用，避免被关停，又能顺便赚一点外快，何乐而不为呢？

    友情提示：这是国外网赚项目，必须要使用境外服务器，否则应该不会产生收益！

    注册网赚平台账号

      要有 Dollar 进账，首先第一步当然是要注册账号啦。博主目前尝试了两个（一个已经无了，仅剩一个）网赚平台，大家可以点击如下链接自行前往注册账号。如果还有其他平台，博主测试后也会在这里更新哒O(∩_∩)O

    Peer2Profit（已无）

    Traffmonetizer

    点击注册账号：Traffmonetizer

    服务器安装 Docker

      因为网赚是通过 Docker 运行，第一步当然是要安装 Docker 啦~如何安装 Docker，官方有详细的说明，跟着做就行啦：Docker Engine installation overview | Docker Documentation

    运行 Docker

      接下来废话不多说，直接放出运行 Docker 的代码，跟着敲就完事啦。

    docker run -d --restart always --name tm traffmonetizer/cli_v2 start accept --token $$your_token_here$$     # amd 架构用这条命令
docker run -d --restart always --name tm traffmonetizer/cli_v2:arm64v8 start accept --token $$your_token_here$$     # arm 架构用这条命令
    #Docker
