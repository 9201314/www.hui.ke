    Ubuntu 优化

    2021-04-07
    2022-11-23
    1 min read
    IT
    Linux
    Hits

      博主比较喜欢新鲜事物，且总认为新版的比旧版的好，于是拿到一个服务器，第一件事就是更新。而对于 Linux 来说，kernel 内核的升级就显得尤为重要。而博主无意间发现了一个对博主超级无敌友好的升级内核方式，简直就是为博主这种人量身定制的——HWE kernel——会自动更新的 kernel.所以赶紧记录下来，同时也附带一些简单的 Ubuntu 优化。

    本文适用系统版本为：搬瓦工 Ubuntu 20.04

    友情提示：搬瓦工 可能需要先翻墙才可访问！

    Time / 时间

    tzselect     # 选时区
cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime     # 改为中国时间

    Source / 源

    nano /etc/apt/sources.list     # 换源
http://archive.ubuntu.com/ubuntu/     # 官方源
sudo apt update     # 更新源

    install HWE-kernel / 安装自动更新内核

    sudo apt install --install-recommends linux-generic-hwe-20.04 -y     # 20.04 用这条命令
sudo apt install --install-recommends linux-generic-hwe-22.04 -y     # 22.04 用这条命令

    uninstall redundant kernels / 卸载多余内核

    dpkg --get-selections | grep linux     # 查看所有内核
sudo apt purge （多余内核）     # 卸载多余内核

    bbr（加速）

    echo net.core.default_qdisc = fq >> /etc/sysctl.conf     # 设置 bbr 队列算法
echo net.ipv4.tcp_congestion_control = bbr >> /etc/sysctl.conf     # 开启 bbr
sysctl -p     # 保存生效
sysctl net.ipv4.tcp_available_congestion_control     # 查看是否有 bbr
lsmod | grep bbr     # 查看 bbr 是否开启
