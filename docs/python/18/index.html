      单项选择题和程序设计考试题

    单项选择题

    1. 自顶向下设计主要由下列哪个语法元素实现？C

      A. 对象

      B. 过程

      C. 函数

      D. 循环结构

    • 函数是自顶向下设计的关键元素，通过定义函数及其参数逐层开展程序设计。

    1. 关于 os.path 子库，以下选项中用来计算相对路径的函数是：A

      A. os.path.relpath(path)

      B. os.path.abspath(path)

      C. os.path.basename(path)

      D. os.path.normpath(path)

    • 顾名思义：os.path.relpath(path) 是 relative path 处理函数。

    1. 以下选项对计算思维本质描述正确的是：B

      A. 抽象和执行

      B. 抽象和自动化

      C. 描述和执行

      D. 计算和思维

    • 计算思维的本质是：抽象和自动化。

    • 抽象一个运算过程，以能够按步骤描述为目标；并利用计算机运算的高速特性自动化执行。

    1. 关于 os 库，以下选项中可以启动进程执行程序的函数是：B

      A. os.process()

      B. os.system()

      C. os.start()

      D. os.run()

    • 除了 os.system()，选项其他函数都不存在。

    1. 关于软件产品和程序的区别，以下描述正确的是：C

      A. 软件产品 = 程序功能 + 速度优化

      B. 软件程序就是软件产品

      C. 软件产品 = 程序功能 + 用户体验

      D. 从程序功能到软件产品的产品化代价很小

    • 产品不仅需要功能，更需要更好的用户体验。往往，产品都需要综合考虑技术功能和人文设计，这源于产品的商业特性。即，商业竞争要求产品不能只关心技术功能，更要关心用户易用和喜好需求。

    1. 关于 Python 的 os 库，以下选项描述正确的是：B

      A. os 库提供了几十个函数，功能比较有限

      B. os 库提供了路径操作、进程管理等若干类功能

      C. os 库是一个第三方库，需要安装后使用

      D. os 库仅适用于 Windows 平台

    • os 库是 Python 重要的标准库之一，提供了几百个函数功能，覆盖与操作系统、文件操作等相关的众多功能。os 库适合所有操作系统。

    1. 以下选项关于计算生态描述错误的是：D

      A. 计算生态主要以开源项目为组织形式

      B. 计算生态存在竞争发展、相互依存和迅速更迭的特点

      C. 计算生态类似自然生态，不受单一组织或人物的控制，形成了技术演化路径

      D. 高质量计算生态需要顶层设计的参与才能保障

    • 计算生态以竞争发展、相互依存和迅速更迭为特点，在开源项目间不存在顶层设计，以类自然界"适者生存"的方式形成技术演进路径。

    1. 关于 Python 第三方库安装方法，以下选项描述错误的是：A

      A. 联系第三方库作者索要安装文件

      B. 访问 UCI 网站下载安装文件

      C. 使用集成安装工具

      D. 使用 pip 命令

    • 请不要联系作者索要第三方库，这不是获取第三方库的合理模式。

    1. 关于用户体验，以下选项描述正确的是：A

      A. 编程只是手段，程序最终为人类服务，用户体验很重要

      B. 用户体验是客观的，完全可以靠技术来实现

      C. 好产品不一定有好的用户体验，关键在于功能创意

      D. 用户体验不重要，只要有程序功能即可

    • 一个提醒进度的进度条、一个永不抛出异常的程序、一个快速的响应、一个漂亮的图标、一个合适尺寸的界面等都是用户体验的组成部分。总的来说，用户体验是一切能够提升程序用户感受的组成。

    1. 关于计算思维，以下选项描述正确的是：C

      A. 计算思维从古至今一直存在

      B. 计算思维关于设计和构造，可以脱离计算机而存在

      C. 计算思维是基于计算机的思维模式

      D. 计算思维是逻辑思维的演进

    • 计算思维是基于计算机的思维模式，计算机出现之前，由于没有快速计算装置，计算所反映的思维模式主要是数学思维，即通过公式来求解问题。当快速计算装置出现后，计算思维才真正形成。

    英文字符的鲁棒输入

    获得用户的任何可能输入，将其中的英文字符进行打印输出，程序不出现错误。

    参考答案

    alpha = []
for i in range(26):
    alpha.append(chr(ord('a') + i))
    alpha.append(chr(ord('A') + i))
s = input()
for c in s:
    if c in alpha:
        print(c, end="")

    答案解析

    注意：这里采用遍历字符的方式实现，通过约束字母表达到鲁棒效果。

    最优解

    for ch in input():
    if ord('a') <= ord(ch) <= ord('z') or ord('A')<=ord(ch)<=ord('Z'):
        print(ch,end='')

    数字的鲁棒输入

    获得用户输入的一个数字，可能是浮点数或复数，如果是整数仅接收十进制形式，且只能是数字。对输入数字进行平方运算，输出结果。

    要求：

    1. 无论用户输入何种内容，程序无错误；
    2. 如果输入有误，请输出"输入有误"。

    参考答案

    s = input()
try:
    if complex(s) == complex(eval(s)):
        print(eval(s)**2)
except:
    print("输入有误")

    答案解析

    complex() 和 complex(eval()) 之间的比较将能够排除非数字类型的输入。

    注意：不能直接使用 eval()，否则，用户可以通过输入表达式（如100**2）输入数字，与要求不同（在实际应用中带来安全隐患）。
